## 📅 Sobre o Evento

O **DEVPIRA Data Saturday Piracicaba 2026** é um evento gratuito voltado para a comunidade de dados, promovido pelo grupo **DEVPIRA** em Piracicaba - SP.

Faz parte da iniciativa global [Data Saturdays](https://datasaturdays.com/), que reúne profissionais e entusiastas da plataforma de dados Microsoft e tecnologias relacionadas.

🔗 [Página oficial do evento](https://eventiza.com.br/evento/devpira-datasaturday-piracicaba-2026)
